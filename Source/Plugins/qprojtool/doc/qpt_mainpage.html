The
Qt Project Tool (qpt)









The “qpt”
(“Qt Project Tool”)  is a command line utility that aims to
extract information from Qt projects (.pro files) making it easy to
use project set of files and settings as input to other command line
tools.


As example other
tools can be: Static Analyzers, Code Counters and others.


When "qpt"
is used with Static Analyzer tools it also can work as a Qt Creator
plugin, making it easy and fast to change source code inside Qt
Creator.
